# Used Car Price Analysis (SQL Project) [Work in Progress]

## Overview
This project focuses on **analyzing used car listings** to uncover key factors affecting car prices using **SQL (MySQL)**. The dataset contains approximately **4,000 rows** and includes information about the car's brand, model, year, mileage, fuel type, transmission, accident history, clean title status, and price.

---

## Dataset
- Source: [Kaggle - Used Cars Dataset by taeefnajib](https://www.kaggle.com/taeefnajib/used-cars)  
- Number of rows: ~4,000  
- Columns:  
  - `brand`, `model`, `model_year`, `milage`, `fuel_type`, `engine`, `transmission`, `ext_col`, `int_col`, `accident`, `clean_title`, `price`  

> **Note:** Irrelevant columns (`ext_col`, `int_col`) were dropped during cleaning.
